[h1]About[/h1]

For Spiritualist MegaCorps, this mod replaces all Managers with Priests/Death Priests/Mortal Initiates.

[h1]Compatibility[/h1]

Not compatible with any other mod that overwrites Temple buildings, including building sort mods. If you have multiple building mods and want this one, put it last.

Not achievement compatible.
